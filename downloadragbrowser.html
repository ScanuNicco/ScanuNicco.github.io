Windows Installation

  • Unzip the downloaded file
  • Open the extracted folder
  • Find Rag Browser Installer.exe and move it to the desktop
  • Close file explorer
  • Run Rag Browser Installer.exe
  • When asked, allow the installer to modify your computer
  • Click on the new desktop shortcut to launch Rag Browser

Troubleshooting the installer

  • Check that the file path of Rag Browser.exe is C:\Users\You user name\Downloads\Rag-Browser\Rag Browser.exe
  • Make sure that file explorer is closed
  • Make sure that you have administrator access

Manual Installation (without installer)

  • Unzip the downloaded file
  • Open the extracted folder
  • Move Rag Browser (not Rag Browser.exe) to the desktop
  • Move the Rag-Browser folder to C:\Program Files
  • Click on the new desktop shortcut to launch Rag Browser

Linux Installation

  • Unzip the downloaded file
  • Run the rag browser executable
  • Pin rag browser to your dock
